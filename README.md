Next came along Data Analytics, took a long walk along that route too and realised looking at spreadsheets all day and making suggestions is really not for me however both these routes did give me a sneakpeak at Python which I really enjoyed. Whichever tools I set off to learn I seem to end up back at Python and that is how a clear path to my future finally came to me.
